# 홈 화면 기획서

## 목적
- 방문자에게 Echo AI의 주요 기능을 간단히 소개합니다.
- 로그인하지 않은 사용자도 서비스를 이해하고 탐색할 수 있도록 안내합니다.

## 섹션 구성
1. **Hero 영역**
   - 서비스 명과 간단한 슬로건 표시
   - "시작하기" 버튼으로 회원가입/로그인 페이지로 이동
2. **주요 기능 소개**
   - 문서 업로드 및 요약
   - 스터디 관리 및 참여
3. **푸터**
   - GitHub 링크 등 외부 링크 제공

## 추후 고려 사항
- 다국어 지원
- 홍보용 이미지 또는 일러스트 추가
